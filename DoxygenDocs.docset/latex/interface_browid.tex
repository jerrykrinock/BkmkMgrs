\subsection{Detailed Description}
A class which has instance variables required to canonically and uniquely identify a \hyperlink{interface_browfile}{Browfile}. 

Conforms to NSCoding and NSCopying so that it can be archived in Core Data stores using the default transformer.

The five string instance variables must consist of only the characters allowed in filenames.

Returns the home path of the Macintosh account in which the receiver's bookmarks reside, if it has a non-nil otherMacAccount.~ Otherwise, returns the current user's home directory. 

\begin{Desc}

The first form will be returned if otherMacVolume is non-nil the second form will be returned if it is nil. \end{Desc}
