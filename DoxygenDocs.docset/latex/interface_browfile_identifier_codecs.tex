\subsection{Detailed Description}
A class which encodes and decodes among browfile identifier strings, browfile identifier parts, and other raw data defining browfile identifiers. 

{\bf Overview} 

A browfile is uniquely identified by either by its {\em identifier\/}, or, equivalently by six {\em parts\/} which compose the identifier.~ This class provides methods for converting back and forth among the identifier, the parts and other raw data from which the parts can be derived. 

All methods provided by this class are factory methods. 

{\bf Specification of a browfile's identifier in terms of its parts} 

An important substring of the identifier is always the {\em exformatDotProfileName\/}. If there is no profileName, exformatDotProfileName is simply the exformat. If there is a profileName, exformatDotProfileName is \char`\"{}exformat.profileName\char`\"{}. Note that the profile name itself may have dots (.) in it.

After that, the form of the browfile identifier depends on the part extoreMedia.~ This table gives that form as a function of extoreMedia: \begin{itemize}
\item {\em BkmmExtoreBuiltIn\/}. ~ This is the basic \char`\"{}default\char`\"{} case. The identifier form is simply the exformatDotProfileName. \item {\em BkmmExtoreOtherHome\/}. ~ There are two possible sub-forms: \begin{itemize}
\item \char`\"{}:volume:account:exformatDotProfileName\char`\"{} \item \char`\"{}:account:exformatDotProfileName\char`\"{} \end{itemize}
The first form is used if the volume root was mounted when the browfile was identified, and the second form is used if the home folder on another Mac was mounted. In the latter case, the volume name is unknown. \item {\em BkmmExtoreStandalone\/}. ~ The identifier form is \char`\"{}file://standalonePath:exformatDotProfileName\char`\"{}. ~ The standalonePath is a full path that itself begins with a slash, and contains slashes in it, up to and including the bookmarks file or folder of bookmarks files. \end{itemize}
