\subsection{Detailed Description}
A managed object which knows static information about an OwnerApp and its exformat.~ To avoid duplication, a browfile can only be obtained using one of the \hyperlink{interface_browfiler}{Browfiler} class methods.~ For the same reason, an extore can only be obtained from one of the browfile instance methods.~ Thus, an extore must always have a browfile. 

GLOSSARY Browser: An installed browser application such as Safari, Camino, etc. Service: Google Bookmarks, del.icio.us, etc. OwnerApp: A Browser (installed application) or a Service (web application) Exformat: The format and structure of the bookmarks in an OwnerApp \hyperlink{interface_browfile}{Browfile}: structure of a profile or account supported by a particular ownerApp, or a builtout browfile. Examples: $>$ structure of a particular Firefox profile $>$ structure of a particular Google Bookmarks account $>$ structure of a particular loose bookmarks file
