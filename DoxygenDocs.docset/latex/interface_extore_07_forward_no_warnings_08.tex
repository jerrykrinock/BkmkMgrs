\subsection{Detailed Description}
Declares that \hyperlink{interface_extore}{Extore} can handle the messages it will forward to its \hyperlink{interface_starker}{Starker}. 

Declaring these in a category is the official trick to eliminate compiler warnings which would otherwise occur where these messages are sent.
