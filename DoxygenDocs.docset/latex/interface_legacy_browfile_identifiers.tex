\subsection{Detailed Description}
A class which converts legacy browfileIdentifier strings into equivalent \hyperlink{interface_browid}{Browid} objects. 

{\bf Overview} 

A browfile is uniquely identified by either by its {\em legacy browfileIdentifier\/}, although other supplemental data (aliasData, browserName) is needed for some types.~ This class provides a method for creating a new \hyperlink{interface_browid}{Browid} object from this identifier and supplemental data.
