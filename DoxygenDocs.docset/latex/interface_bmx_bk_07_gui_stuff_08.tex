\subsection{Detailed Description}
A category on \hyperlink{class_bmx_bk}{BmxBk} that provides methods which elements in the GUI need for binding. 

This is just to keep BmxBk.m from getting too long and cluttered with trivial stuff. Many of the simple methods which are accessed via bindings are not declared.
