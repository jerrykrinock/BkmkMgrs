\subsection{Detailed Description}
Formal protocol for an object which can contain Starks in a managed object context, and has a \hyperlink{interface_starker}{Starker} available. 

Conforming classes will provide methods to

Access the \hyperlink{interface_starker}{Starker} (of course!) Answer questions regarding placing additional Starks in the receiver Identify the class's \hyperlink{interface_browfile}{Browfile} (optional)

This protocol was established to express the commonality between the internal representation of Starks in a \hyperlink{class_bookshelf}{Bookshelf} and their external representation in an \hyperlink{interface_extore}{Extore}. \hyperlink{class_bookshelf}{Bookshelf} and \hyperlink{interface_extore}{Extore} adopt this protocol.
