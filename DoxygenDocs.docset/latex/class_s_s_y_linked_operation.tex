\subsection{Detailed Description}
A subclass of NSOperation which allows operations to be skipped, by using the value of an error. 

The error is the value for keyconstKeyLinkedOperationError in the receiver's 'info'. If the -code of this error equals or exceeds the receiver's 'errorThreshold', the operation becomes a no-op.

The comparison which decides whether or not to execute the method is done in the -main implementation which is not ordinarily overridden if the operation is created by SSYOperationLinker.
