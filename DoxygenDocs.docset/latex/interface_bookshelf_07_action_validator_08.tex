Calculates current parameters (enabled, target, action, title) for a menu item which could visit the receiver's currently-selected starks with a given url temporality. 

\begin{Desc}
\item[Parameters:]
\begin{description}
\item[{\em urlTemporality}]The temporality (prior URL, current URL, or suggested URL) of the url to be used in the hypothetical visit. \item[{\em target\_\-p}]On return, if non-NULL, points to the target of the menu item. \item[{\em action\_\-p}]On return, if non-NULL, points to the action of the menu item.~ This will be NULL if the menu item should not be enabled. \item[{\em title\_\-p}]On return, if non-NULL, points to the title of the menu item. \end{description}
\end{Desc}
\begin{Desc}
\item[Returns:]YES if the menu item should be enabled, NO otherwise. \end{Desc}
