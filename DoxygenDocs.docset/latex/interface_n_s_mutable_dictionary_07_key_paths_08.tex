\subsection{Detailed Description}
NSMutableDictionary can read values from nested dictionaries with -\mbox{[}NSObject valueForKeyPath:\mbox{]}, but not set such values.~ This category adds the missing setValue:forKeyPath: method, and methods to mutate arrays located down key paths.~ Each method also has a companion which takes a \char`\"{}key path array\char`\"{}, which is a an array of string keys, instead of a key path, which is an equivalent dot-separated string. 

This class is a clone of NSUserDefaults (KeyPaths).
