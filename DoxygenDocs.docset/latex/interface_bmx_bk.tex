\subsection{Detailed Description}
Persistent Document of the \char`\"{}Bookmarks Book\char`\"{}. 

Considered using just one notification instead of seven, using userInfo to identify which timestamp to update. But since I need to coalesce on name to reduce the number of notifications, this would, for example, coalesce 'sorted' and 'unsorted' into one notification instead of having the 'unsorted' be first in/out of the queue and 'sorted' last in/out of the queue as desired.
