\subsection{Detailed Description}
An NSMenu helper which can add contextual menu items for a \hyperlink{class_stark}{Stark} context, or return an entire prefabricated menu containing all avalailable items. 

Items in menus built using the methods of this class are disabled as needed when they are created, by setting a target, and by setting their action to NULL.~ Do not implement -validateMenuItem: or -validateUserInterfaceItem for these menu items.

Most of the methods take, as arguments, the array of starks which would be involved if the menu command is clicked.~ For convenience, these arrays are passed through to the action methods when a menu item is clicked as the -representedObject.~ (The menu item is the 'sender' parameter passed to the action method, so that the action method may recover the array by sending \mbox{[}sender representedObject\mbox{]})
