\subsection{Detailed Description}
A class for programmatically conducting a Spotlight search for a named file, returning an array of paths. 

NSObject \{ NSMetadataQuery $\ast$query ; id callbackTarget ; SEL callbackSelector ; \}  Two class methods is exposed. findPathsWithPredicate:callbackTarget:callbackSelector: is the general, base method. findPathsWithFilename:callbackTarget:callbackSelector: is a wrapper around the base method which shows how to construct predicates for a the search \hyperlink{interfaceyou}{you} need. For a complete list of attributes, consult Apple's \char`\"{}MD Item Reference\char`\"{} documentation and scroll down to \char`\"{}File System Metadata Attribute Keys\char`\"{}.

The class method creates an instance which self-destructs after the search is complete. This class requires Mac OS 10.5 or later.

Finds paths of files with a given predicate, by doing a Spotlight search, i.e. using NSMetadaQuery. 

Runs asynchronously and returns an NSArray of NSString objects to a specified callback target object, via a specified callback selector. \begin{Desc}
\item[Parameters:]
\begin{description}
\item[{\em predicate}]The predicate used to filter items, or qualify results, in the search. \item[{\em callbackTarget}]The object which will receive the callback. \item[{\em callbackSelector}]The selector in the callback target which will be invoked. This selector should take one argument, an NSArray, and should return void. \end{description}
\end{Desc}
