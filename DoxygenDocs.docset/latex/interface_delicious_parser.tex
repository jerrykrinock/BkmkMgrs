Parses given data downloaded from del.icious, obtaining, optionally: posts and tags, dates and/or bundles. 

\begin{Desc}
\item[Parameters:]
\begin{description}
\item[{\em data}]\item[{\em doStarks}]If YES, bookmarks and tags are parsed from data and inserted into the localMoc of the extore. \item[{\em doDates}]Switches on a function which is not implemented at this time. \item[{\em doBundles}]Switches on a function which is not implemented at this time. \item[{\em extore}]The extore into whose localMoc any bookmarks and tags found will be inserted as starks, and whose browfile's browid the exid of each inserted stark will be associated with. \item[{\em lastUpdateTime}]Pointer to an NSDate$\ast$ or NULL.~ If not NULL, and a last update time is found while parsing the data, an NSDate representing this time will be assigned to this pointer. \item[{\em error}]Pointer to an NSError$\ast$ or NULL.~ If not NULL, and an error occurs, an NSError explaining the error will be assigned to this pointer. \end{description}
\end{Desc}
\begin{Desc}
\item[Returns:]YES if the operation completes with no error, otherwise NO. \end{Desc}
