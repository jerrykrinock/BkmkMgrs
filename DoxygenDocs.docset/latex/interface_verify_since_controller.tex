\subsection{Detailed Description}
Window controller subclass for the sheet which asks user which bookmarks -- since last when verified -- to verify. 

Note: In the nib containing the sheet/window, make sure that in the window attribute \char`\"{}Visible on Launch\char`\"{} is NOT checked. If it is, the window will display immediately as a freestanding window instead of the next line attaching it to hostWindow as a sheet.
