This is for representing data in a filename.~ Although the Finder allows \hyperlink{interfaceyou}{you} to use a slash in filenames, it converts them to a colon since unix does not allow slashes.~ But Finder does not allow colons.~ This could get very confusing, so I'd rather not use slashes in filenames.
