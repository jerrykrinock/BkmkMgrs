# Getting Started with Markster <img src="images/Markster.png" class="whapp" /> [startMarkster]

Follow the steps below.  If you click a link for more detail, use the Back (<) button in the toolbar above to return here.

* [Copy any existing bookmarks.](oneTimeIxports) into your [Collection]().

* Review the following facilities which you may use to *land* (add) new bookmarks into Markster, or visit bookmarks currently in Markster.  Decide which facilities you would like to use.  Some of them require some setting up.

&#160; &#160; &#160; &#160; Facility                                  |   Landing   |  Visiting   | Works in browsers
:---------------------------------------------------------------------|:-----------:|:-----------:|:------------------------
&#160; &#160; &#160; &#160; [Keyboard Shortcuts](globShorts)          |      ✔      |      ✔      | All [readable browser apps](readableBrowserApp)
&#160; &#160; &#160; &#160; [Menu Extra](bkmxMenuExtra)               |      ✔      |      ✔      | All [readable browser apps](readableBrowserApp)
&#160; &#160; &#160; &#160; [Dock Menu](bkmxDockMenu)                 |      ✔      |      ✔      | All [readable browser apps](readableBrowserApp)
&#160; &#160; &#160; &#160; [Browser Menu Items](browserWidgets)      |      ✔      |             | Firefox only
&#160; &#160; &#160; &#160; [Browser Bar Button](browserWidgets)      |      ✔      |             | Brave, Chrome, Edge, Orion, FreeSMUG's Chromium, Canary only
&#160; &#160; &#160; &#160; [Bookmarklet](bookmacsterizer)            |      ✔      |             | All browsers
&#160; &#160; &#160; &#160; [Scripting Markster](scriptingBkmx)       |      ✔      |             |

* For each facility you'd like to use, click its name in the table above for setup instructions.

* Designate a default folder for [New Bookmark Landing]().

* If you don't want [the sound when landing new bookmarks](landingSoundFx), switch off that preference.

* If you don't want to use the default web browser you have already set in macOS, set the [default web browser for visiting from Markster](doxtusVisitWhich).

* Consider whether or not you'd like to [hide Markster](hideBkmx).
